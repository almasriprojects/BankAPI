Chase Bank Statement Parser API

Welcome to the Chase Bank Statement Parser API.

This service processes Chase Bank statements (PDF) using OpenAI, converting them into structured JSON data.

Quick Start

POST /upload-statement

Upload and process a bank statement PDF file.

Key Features

  • PDF statement upload and processing
  • OpenAI-powered document parsing
  • Structured JSON output
  • Transaction categorization
  • Spending analysis
  • Balance validation

Requirements

  • OpenAI API key (set in environment variables)
  • PDF file format
  • Maximum file size: 20MB
